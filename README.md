# Todo List

This is a simple todo list project built with javascript

## Structure

- node_modules
- main.js
- static
  - dist
  - scss
  - index.html

## Instructions

Just use [live server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) or run `node main.js` it will hosted on localhost:3000 in your browser
